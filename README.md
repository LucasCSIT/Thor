# Thor
Thor is a **plugin** for Minecraft: Java Edition v1.21.11 that grants the wielder of a diamond axe some of Thor's abilities!

## Requirements
* Minecraft: Java Edition v1.21.11
* Paper 1.21.11

## Controls
* **Left Click**: Strike lightning at the targeted location
* **Right Click**: Strike lightning at all enemies near you.
* **Sneak**: Strike lightning in front, behind, to the left, and to the right of you!
* **Drop Item**: Throw your axe to strike lightning at it's landing location. The axe will return back to your inventory.

## Installation
1. Download [PaperMC v1.21.11](https://papermc.io).
2. Download the plugin visiting the [official Hangar page](https://hangar.papermc.io/Sharp/Thor).
3. Once you download the .jar file, drag the file into your `plugins` folder that was a part of your PaperMC download.
4. Start the PaperMC server by doing `cd /path/to/server/` and executing `java -Xms4G -Xmx4G -jar paper.jar --nogui` (both in a PowerShell Terminal).
   - Visit [Getting Started](https://docs.papermc.io/paper/getting-started/) on PaperMC's site for more info.
5. Once the server is started, enter your port as a new server or direct connection in the Multiplayer menu.
6. Get a diamond axe and enjoy!
