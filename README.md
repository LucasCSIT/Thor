# Thor
Thor is a **plugin** for Minecraft: Java Edition v1.21.11 that grants the wielder of a diamond axe some of Thor's abilities.

## Requirements
* Minecraft: Java Edition v1.21.11
* Paper 1.21.11

## Features
* **Call down lightning** exactly where you're looking.
* **Strike lightning** on nearby enemies.
* **Surround yourself** with lightning for a short time.
* **Throw your axe** to strike lightning from a distance in style.

## Controls
* **Left Click**: Strike lightning at the targeted location.
* **Right Click**: Strike lightning at all nearby enemies.
* **Sneak**: Strike lightning in front, behind, to the left, and to the right of you.
* **Drop Item**: Throw your axe to strike lightning at its landing location; it will return to your inventory.

## Installation
1. Download [PaperMC v1.21.11](https://papermc.io).
2. Download the plugin from the [official Hangar page](https://hangar.papermc.io/Sharp/Thor).
3. Drag the `.jar` file into your PaperMC server's `plugins` folder.
4. Start the PaperMC server by doing `cd /path/to/server/` and executing `java -Xms4G -Xmx4G -jar paper.jar --nogui` in the Terminal.
   - See [PaperMC Getting Started](https://docs.papermc.io/paper/getting-started/) for more info.
5. Join the server, get a diamond axe, and enjoy!
