# Thor
Thor is a **plugin** for Minecraft: Java Edition v1.21.11 that grants the wielder of a diamond axe some of Thor's abilities!

# Download
Please click __here__ to visit the official Thor page on Hangar!

## Requirements
* Minecraft: Java Edition v1.21.11
* Paper 1.21.11
* Gradle

## Controls
* **Left Click**: Strike lightning at the targeted location
* **Right Click**: Strike lightning at all enemies near you.
* **Sneak**: Strike lightning in front, behind, to the left, and to the right of you!
* **Drop Item**: Throw your axe to strike lightning at it's landing location. The axe will return back to your inventory.
